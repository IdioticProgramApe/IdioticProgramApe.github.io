## Statistics Panel

How to access

- **UE5**: Tools -> Audit -> Statistics

- **UE4:** Windows > Statistics



Available stats regarding to the current loaded level:

| Stats Name                | Notes                                                        |
| ------------------------- | ------------------------------------------------------------ |
| **Primitives Stats**      | - check size to find the heaviest asset<br />- amount of actors/instances referenced |
| **Texture Stats**         |                                                              |
| Cooker stats              |                                                              |
| Lighting Build Stats      |                                                              |
| Static Mesh Lighting Info |                                                              |
| Shader Cooker Stats       |                                                              |



All the statistics column name explanations are specified in **PrimitiveStats.h**
